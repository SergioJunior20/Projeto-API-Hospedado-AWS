# 🚀 TB2 - AWS | Acesso a Recursos de Rede em Sistemas Distribuídos

Projeto acadêmico desenvolvido para a disciplina de **Sistemas Distribuídos** com foco em desenvolvimento backend, segurança, banco de dados e deploy em nuvem utilizando AWS.

---

# 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **API RESTful CRUD**, permitindo operações de criação, leitura, atualização e remoção de dados, utilizando conceitos de sistemas distribuídos e computação em nuvem.

A aplicação foi hospedada na **AWS EC2**, com configuração de segurança, autenticação e integração com banco de dados MongoDB.

---

# 🛠️ Tecnologias Utilizadas

| Tecnologia | Utilização |
|------------|------------|
| **Node.js** | Backend da aplicação |
| **JavaScript** | Desenvolvimento da API |
| **MongoDB** | Banco de dados NoSQL |
| **JWT** | Autenticação e segurança |
| **Postman** | Testes das rotas CRUD |
| **AWS EC2** | Hospedagem da aplicação |
| **SSH** | Acesso remoto seguro |
| **.env** | Variáveis de ambiente |

---

# ⚙️ Funcionalidades

✅ Cadastro de registros  
✅ Listagem de dados  
✅ Atualização de informações  
✅ Remoção de registros  
✅ Autenticação via token  
✅ Proteção de rotas privadas  
✅ Integração com MongoDB  
✅ Deploy em ambiente cloud AWS  

---

# 🔐 Segurança

O projeto utiliza mecanismos de segurança para proteção da aplicação e acesso remoto:

- Autenticação com JWT
- Basic Auth / Token próprio
- Configuração de Security Groups
- Acesso seguro via SSH
- Variáveis de ambiente com `.env`

---

# ☁️ Infraestrutura AWS

A aplicação foi hospedada utilizando:

- **AWS EC2**
- Configuração de firewall
- Chaves SSH para acesso remoto
- Instância Linux para execução do backend

---

# 🧪 Testes da API

Todos os endpoints foram testados utilizando o **Postman**, garantindo o funcionamento correto das operações CRUD e autenticação.
